### Prerequisites

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/vedxbuilder.git
   cd vedxbuilder
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
